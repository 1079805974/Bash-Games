Bash-Games
===========
Linux终端下的小游戏，和小东西，包括 俄罗斯方块，贪吃蛇，屏保时钟，翻译脚本。。。
所有的方向按键都是按照 vim ‘hjkl’ 来的！ 
Some linux terminal games, include Tetris, Snake, coding by bash shell

数字时钟 (Clock)
================
ClockSaver.sh 利用了 logkey s这个工具(需要root权限), 所以可以达到屏保的目的，无操作时显示数字时钟
